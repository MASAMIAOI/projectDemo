# sms 短信发送相关

~~~
AccessKey ID  ---------- 点击用户头像， AccessKey 管理获取
AccessKey Secret ---------- 点击用户头像， AccessKey 管理获取
PhoneNumbers ---------- 电话号
SignName ---------- 短信签名名称：开启短信服务，进入控制台点击左侧国内消息添加
TemplateCode ---------- 短信模板ID：开启短信服务，进入控制台点击左侧国内消息添加
TemplateParam ---------- 短信模板中的参数，json 类型 {"code":6379}
~~~
